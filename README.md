#   Mon API FastAPI

projet realise pour apprendre les bases des **API REST** et de l'architecture **MVC**.

## Installation
1. Installer les dependances: " pip install fastapi, pip install uvicorn"
2. Lancer le serveur : "uvicorn nom_fichier:app --reload"

## Fonctionnalites
- Creation d'items (POST)
- Lecture de la liste (GET)
- Lecture d'un item par ID (GET)
- Documentation automatique sur "/docs"
